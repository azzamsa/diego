# Diego

Diego is a pelican port of Hugo theme with the same name. It's clean, fast and
responsive.

## Feature

- Translatable pages
- Comments
- Language switching via pelican's i18n_subsites

To see how it looks, visit [my website][my-web] or its [configuration][sc]
